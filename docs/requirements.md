# 요구사항 체크리스트

## Weather (날씨)

- [x] 첫 진입 시 사용자의 현재 위치를 감지하고 해당 위치의 날씨 표시
- [x] Open API를 통해 날씨 정보를 가져와 화면에 표시
  - [x] 위치(장소) 이름
  - [x] 현재 기온
  - [x] 당일 최저 기온
  - [x] 당일 최고 기온
  - [x] 시간대 별 기온
- [x] 선택한 장소의 날씨 정보가 없을 경우 다음 문구 표시: "해당 장소의 정보가 제공되지 않습니다."

## Search (검색)

- [x] 대한민국 한정으로 주소 검색 (시/군/구/동 단위 모두 입력 가능)
  - [x] 제공된 `korea_districts.json` 활용 (필요 시 구조 변환 가능)
- [x] 검색어 입력 시 매칭되는 장소 리스트 표시
- [x] 리스트에서 장소 선택 가능

## Favorites (즐겨찾기)

- [x] 검색한 장소를 즐겨찾기에 추가/삭제
- [x] 즐겨찾기 최대 6개 제한
- [x] 즐겨찾기 항목은 카드 UI로 표시
  - [x] 장소 이름(별칭) 표시 및 수정 가능
  - [x] 카드에 현재 날씨 + 당일 최저/최고 기온 표시
- [x] 즐겨찾기 카드 클릭 시 상세 페이지로 이동
  - [x] 상세 페이지에 날씨 섹션의 전체 정보 표시
